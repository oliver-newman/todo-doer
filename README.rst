Todo Doer
=============

Lightweight command-line todo manager.
-------------------------------------------

Usage:

::

    $ todo-doer add Finish all my todos
    $ todo-doer add Make sure I finished all my todos
    $ todo-doer
    1. Finish all my todos
    2. Make sure I finished all my todos
    $ todo-doer do 2
    $ todo-doer
    1. Finish all my todos
    $ todo-doer -a
    1. Finish all my todos
